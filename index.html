Difference Between 2D Arrays and Matrices in NumPy

1. Definition

  • Matrix (numpy.matrix): A specialized type for linear algebra. Always two-dimensional.
  • 2D Array (numpy.array): A general container that can have any number of dimensions.
import numpy as np

# Creating a matrix
mat = np.matrix([[1, 2], [3, 4]])

# Creating a 2D array
arr = np.array([[1, 2], [3, 4]])

print(mat.shape)  # (2, 2)
print(arr.shape)  # (2, 2)

2. Multiplication

  • Matrix: * performs matrix multiplication
  • 2D Array: * performs element-wise multiplication
import numpy as np

mat = np.matrix([[1, 2], [3, 4]])
arr = np.array([[1, 2], [3, 4]])

# Matrix multiplication
print(mat * mat)  # Matrix multiplication
# [[7 10]
#  [15 22]]

# Array multiplication
print(arr * arr)  # Element-wise
# [[1 4]
#  [9 16]]

# Matrix multiplication with arrays
print(arr @ arr)  # Matrix multiplication
# [[7 10]
#  [15 22]]

3. Accessing Rows/Columns

  • Matrix: Returns a matrix
  • 2D Array: Returns a 1D array
import numpy as np

mat = np.matrix([[1, 2], [3, 4]])
arr = np.array([[1, 2], [3, 4]])

print(mat[0])  # [[1 2]] (Matrix)
print(arr[0])  # [1 2] (1D Array)

4. Mathematical Operations

  • Matrix: Has .I property for inverse and ** for matrix power
  • 2D Array: Uses np.linalg.inv() for inverse and ** for element-wise power
import numpy as np

mat = np.matrix([[1, 2], [3, 4]])
arr = np.array([[1, 2], [3, 4]])

# Inverse
print(mat.I)  # Matrix inverse
print(np.linalg.inv(arr))  # Array inverse

# Power
print(mat**2)  # Matrix power
print(arr**2)  # Element-wise power

5. Which One to Use?

  • Matrix: Good for linear algebra, but less flexible
  • 2D Array: Recommended for most use cases in NumPy, more versatile
import numpy as np

# Modern NumPy recommendation
# Prefer arrays over matrices
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Linear algebra operations
result = A @ B  # Matrix multiplication
inverse = np.linalg.inv(A)  # Matrix inverse
